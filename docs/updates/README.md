# Dated Update Logs

이 폴더는 날짜별 작업 로그를 누적 저장하는 문서 폴더다.

규칙:

- 파일명은 `YYYY-MM-DD.md`
- 각 문서에는 그 날짜에 반영한 수정, 근거, 남은 문제, 다음 단계만 기록
- 기존 요약 문서([change.md](/home/deepblue/AV_Drone/docs/change.md), [problem.md](/home/deepblue/AV_Drone/docs/problem.md))는 상태 문서로 유지하고, 이 폴더는 작업 이력 문서로 누적

현재 최신 로그:

- [2026-03-13.md](/home/deepblue/AV_Drone/docs/updates/2026-03-13.md)
- [2026-03-19.md](/home/deepblue/AV_Drone/docs/updates/2026-03-19.md)
- [2026-03-20.md](/home/deepblue/AV_Drone/docs/updates/2026-03-20.md)
