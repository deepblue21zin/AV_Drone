## Runtime Findings
- `sim` 컨테이너는 올라오고 `Startup script returned successfully`도 확인했다.
- `ros_states`는 Flask 누락으로 죽던 문제를 확인했고, 의존성과 런타임 설치로 복구했다.
- 현재 `sim` 로그에는 `ERROR [simulator_mavlink] poll timeout 0, 22`가 반복된다.
- 자율주행 노드가 떠 있어도 다음 입력이 비어 있으면 미션은 `WAIT_STREAM`에서 벗어나지 못한다.
  - `/mavros/state` 는 들어오더라도 `connected: false` 상태일 수 있음
  - `/mavros/local_position/pose` 샘플이 오지 않는 경우가 있음
  - `/drone1/scan` 샘플이 오지 않는 경우가 있음
- 따라서 현재 주요 미해결 항목은 UI가 아니라 simulator/FMU/pose/scan 실시간 체인이다.
