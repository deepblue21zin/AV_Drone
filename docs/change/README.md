# Change Log

이 폴더는 작업 중 발생한 변경 사항을 날짜별로 누적 기록하는 공간입니다.

규칙:
- 파일명은 `YYYY-MM-DD.md` 형식을 사용합니다.
- 각 파일에는 그날의 변경 내용, 이유, 검증 상태, 남은 이슈를 기록합니다.
- 이 기록은 작업 메모 성격의 로그이며, Git 히스토리를 대체하지 않습니다.
